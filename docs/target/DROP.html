DROP

DROP

语法 

# 删除数据库
drop database dbname;    
# 删除表
drop table table_name;    
# 删除索引
drop index index_name on table_name;

:warning: 特别注意:

​ 删库删表需要特别谨慎！在生产环境下几乎不会进行删库删表操作。
